# Codebase Stack

## Languages & Runtime
- **Runtime**: Node.js
- **Language**: JavaScript (ES6+)
- **Framework**: [Next.js](https://nextjs.org/) (v16.1.1)
- **Library**: [React](https://reactjs.org/) (v19.1.0)

## Frontend Frameworks & Libraries
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v4.1.12) with PostCSS
- **Animations**:
  - [GSAP](https://greensock.com/gsap/) (v3.14.2)
  - [Framer Motion](https://www.framer.com/motion/) (v12.23.12)
  - [React Type Animation](https://www.npmjs.com/package/react-type-animation) (v3.2.0)
- **3D / Visuals**:
  - [Three.js](https://threejs.org/) (v0.179.1)
  - [@react-three/fiber](https://github.com/pmndrs/react-three-fiber) (v9.3.0)
  - [@react-three/drei](https://github.com/pmndrs/drei) (v10.7.2)
  - [tsparticles](https://particles.js.org/) (v3.9.1)

## Development Tools
- **Linting**: Next.js ESLint
- **Package Manager**: npm
- **PostCSS**: v8.5.6

## Configuration Files
- `package.json`: Dependencies and scripts
- `next.config.mjs`: Next.js configuration
- `jsconfig.json`: Path aliases (`@/*` -> `./src/*`)
- `postcss.config.mjs`: PostCSS configuration
- `globals.css`: Global styles and Tailwind configuration
